# References

## Reference Data Requirements

The variant calling pipeline requires the following reference data files:

### Human Reference Genome
- **GRCh38/hg38**: Recommended reference genome assembly
- **GRCh37/hg19**: Also supported for legacy datasets
- **Format**: FASTA format (.fa, .fasta)

### Required Index Files
- **FASTA index**: `.fai` file (created by `samtools faidx`)
- **Sequence dictionary**: `.dict` file (created by GATK or Picard)
- **BWA-MEM2 indices**: Generated by `bwa-mem2 index`

### Known Variants for BQSR
- **dbSNP**: Common SNPs for base quality score recalibration
- **Mills and 1000G indels**: High-confidence indel sites
- **Format**: VCF format, bgzip compressed with tabix index

- Creates 1kb test reference with synthetic variants

## Best Practices
1. Always use the same reference genome consistently across all samples
2. Ensure all index files are created from the same reference FASTA
3. Use appropriate reference version for your species and analysis
4. Verify file integrity with checksums when available
